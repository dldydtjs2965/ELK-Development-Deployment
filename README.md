## 📖 스터디룰 
- 불참 3회시 스터디 강퇴
- 사전고지없이 당일 불참시 불참 2회 카운트
- 스터디 절반 이상 진행 후 참여시 불참 카운트
- 발표 자료가 부실할 경우 재발표를 진행하거나 불참 카운트
- 스터디 진행시 질문 1회 이상 하는 것을 권장합니다
